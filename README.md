#Farm Simulator

##Description
Farm Simulator is an accessible, interactive, and user-friendly menu-based simulator game.  
This project is currently a work in progress and is my first programming project.

##purpose
The goal of this project is to build a simple farming simulation that can be easily used with keyboard navigation and screen readers.

##features
- Menu-based navigation
- Accessible design for screen reader users
- Simple and clear gameplay flow
- save slots

##Status
Work in progress. Features will be added gradually.
